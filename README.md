# AI-Internship-Tasks-2

This repository contains a collection of Artificial Intelligence and Machine Learning tasks developed during the internship at **DevelopersHub**. The projects cover various domains including Natural Language Processing (NLP), Large Language Model (LLM) orchestration, and multi-label text classification.

## 📁 Repository Structure

The repository consists of the following project notebooks:

| File Name | Description | Tech Stack / Models |
| :--- | :--- | :--- |
| `Chatbot_Using_LangChain.ipynb` | A contextual chatbot built using LangChain capable of handling user queries with memory. | LangChain, Python, LLMs |
| `News_Classifier_BERT.ipynb` | A fine-tuned BERT model designed to classify news articles into their respective categories. | BERT, Hugging Face, PyTorch/TensorFlow |
| `Support_Ticket_Tagging.ipynb` | An automated classification system that assigns relevant tags to incoming customer support tickets. | NLP, Scikit-Learn / Deep Learning |

---

## 🚀 Projects Overview

### 1. Chatbot Using LangChain
* **Objective:** Build an intelligent, conversational chatbot.
* **Key Features:** Implements conversational memory, prompt structuring, and chains to simulate realistic, multi-turn human interactions.

### 2. News Classifier (BERT)
* **Objective:** Classify news headlines or articles into distinct categories (e.g., Tech, Sports, Business).
* **Key Features:** Leverages a pre-trained **BERT (Bidirectional Encoder Representations from Transformers)** model, fine-tuned on a custom news dataset for high-accuracy text classification.

### 3. Support Ticket Tagging
* **Objective:** Streamline customer service workflows by automatically categorizing support tickets.
* **Key Features:** Processes raw textual tickets, cleans the text, and applies classification algorithms to route or tag tickets based on the user's issue.
